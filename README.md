# 9-Day Ascend - JEE Main Study Planner

A modern, dark-themed study planner web app for JEE Main aspirants following a structured 9-day intensive preparation plan.

✨ Features
🎯 9-Day Structured Plan
Pre-loaded with comprehensive schedule covering Electrostatics, Calculus, Coordination Compounds, Modern Physics, and more across morning, afternoon, and night sessions.

📚 Subject Organization
Separate tabs for Physics, Mathematics, and Chemistry with Mathematics positioned in the middle for intuitive navigation.

📊 Dual Progress Tracking
• Overall progress card with global completion percentage
• Subject-wise progress card for active subject
• Individual progress bars with percentages for each subject

✅ Lecture & PYQ Tracking
Track both lecture completion and solved previous year questions separately for every chapter.

🔄 Expandable Day Cards
Click any day to expand and view detailed session details with checkboxes.

➕ Custom Chapters
Add unlimited extra topics to any subject with full tracking capabilities.

💾 Auto-Save
All progress persists in browser localStorage - close and reopen without losing data.

🎨 Modern Dark UI
Radial gradient background, glassmorphic cards, smooth animations, and responsive design.

🚀 Quick Start
→ Open index.html in any modern browser
→ Select a subject tab (Physics | Mathematics | Chemistry)
→ Expand day cards and check off completed tasks
→ Add custom chapters using the input field
→ Track progress through dual progress indicators

🛠️ Built With
• HTML5 - Semantic structure
• CSS3 - Modern styling with gradients and animations
• JavaScript (ES6+) - Dynamic rendering and state management
• Font Awesome 6 - Icon library
• Google Fonts (Inter) - Typography
• LocalStorage API - Client-side data persistence

💾 Data Persistence
• All progress automatically saves to localStorage. Use reset buttons to manage your data:
• Reset all checkboxes - Clear progress, keep chapters
• Wipe all progress - Restore default 9-day plan
